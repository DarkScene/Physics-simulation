# Physics-simulation
교내 '실감물리시물레이션' 실습 결과물

: 강체동역학(충돌감지 포함), 연체동역학, 유체동역학과 같은 단순한 특정 물리 시스템을 최대한 시뮬레이션 하는 물리 엔진 개발 수업

OpenGL을 이용 / C++ 사용 / visualstudio2017 사용
